# Code of Conduct

Nemiga Tower aims to be a respectful, low-friction public repository.

## Expected Behavior

- Be clear, constructive, and specific in issues and pull requests.
- Assume good intent and focus feedback on the work, not the person.
- Prefer actionable reproduction steps, screenshots, and concrete suggestions.

## Unacceptable Behavior

- Harassment, insults, or demeaning language
- Trolling, bad-faith disruption, or spam
- Publishing private, sensitive, or security-relevant information in public threads

## Scope

This code of conduct applies to repository discussions, issues, pull requests, and other project communication spaces controlled by the maintainer.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainer privately using the public contact method listed on the GitHub profile. Security-sensitive reports should follow [`SECURITY.md`](./SECURITY.md).

## Enforcement

The maintainer may edit, hide, lock, or remove content that violates this code of conduct and may limit further participation when necessary to protect the project space.
