# Руководство по Дизайн-Системе: Neo-Corporate Excellence

Данный документ определяет визуальный язык и принципы проектирования для высокотехнологичной бизнес-среды. Наша цель — выйти за рамки стандартных интерфейсов, создавая ощущение «цифрового консьержа»: премиального, сдержанного и интуитивно понятного.

## 1. Creative North Star: «Архитектурная Глубина»

Концепция системы строится на **Neo-Corporate Glassmorphism**. Мы не просто рисуем интерфейс, мы проектируем пространство. 
- **Асимметрия и Ритм:** Вместо жесткой, монотонной сетки используйте Bento Grid с варьирующимися пропорциями. Это создает визуальный интерес и позволяет выделять приоритетный контент через масштаб.
- **Многослойность:** Интерфейс должен ощущаться как набор тонких листов матового стекла, парящих над бездной темного обсидиана.
- **Эффект Присутствия:** Мы отказываемся от «плоского» дизайна в пользу глубины, достигаемой за счет тональных переходов и мягкого свечения.

## 2. Цветовая Палитра и Светопись

Основа системы — глубокий `background` (#10131a) и акцентный благородный `primary` (Gold #D4AF37).

### Правило «No-Line» (Без линий)
**Строго запрещено** использование сплошных 1px границ для разделения секций. Границы контента определяются исключительно:
1.  Сменой тона поверхностей (например, карточка `surface_container_low` на фоне `surface`).
2.  Мягким свечением (Bloom) или эффектом Glassmorphism.

### Иерархия поверхностей и вложенность
Для создания физического ощущения глубины используйте систему тонального наслоения:
- **Основа:** `surface` (#10131a) — нижний слой.
- **Секции:** `surface_container_low` (#191b23).
- **Карточки Bento:** `surface_container` (#1d1f27) с прозрачностью `rgba(255, 255, 255, 0.03)` и `backdrop-filter: blur(20px)`.
- **Активные элементы:** `surface_container_highest` (#32353d) для выделения самого важного.

### Signature Textures
Для главных CTA-кнопок и заголовков Hero-секций используйте градиент от `primary` (#f2ca50) к `primary_container` (#d4af37). Это придает золоту «металлический» отлив и исключает визуальную плоскость.

## 3. Типографика: Редакционный Стиль

Мы используем сочетание **Manrope** (Display/Headline) для авторитетности и **Inter** (Body/Labels) для функциональности.

*   **Display LG (Manrope, 3.5rem):** Для ключевых цифр и имиджевых заголовков.
*   **Headline MD (Manrope, 1.75rem):** Заголовки блоков в Bento Grid.
*   **Body LG (Inter, 1rem):** Основной текст. Увеличенный межстрочный интервал (1.6) для премиального ощущения «воздуха».
*   **Label MD (Inter, 0.75rem):** Техническая информация, всегда в Caps Lock с трекингом +5-10% для «архитектурного» вида.

## 4. Глубина и Элементарные Частицы

### Принцип наслоения (Layering)
Вместо теней используйте разницу в яркости токенов. Поместите карточку `surface_container_lowest` на секцию `surface_container_low`, чтобы создать естественный визуальный провал или подъем.

### Ambient Shadows (Окружающий свет)
Если элементу необходимо «взлететь» (например, модальное окно):
- Использовать `shadow` с размытием 40-60px.
- Цвет тени: тинтированный `surface_container_lowest` с прозрачностью 40%, а не черный. Это создает эффект мягкого рассеивания света.

### Ghost Borders (Призрачные границы)
Если граница необходима по требованиям доступности, используйте токен `outline_variant` с непрозрачностью не более 10-20%. Это создает намек на форму, не перегружая глаз.

## 5. Компоненты

### Кнопки (Buttons)
*   **Primary:** Фон — градиент Gold, текст — `on_primary`. Эффект: внешнее свечение (box-shadow) цветом `primary` с размытием 15px при наведении.
*   **Secondary:** Прозрачный фон, «Ghost Border» 1px, текст `primary`.
*   **Tertiary:** Без границ, только текст в цвете `on_surface_variant`.

### Bento-карточки
*   **Скругление:** Всегда `xl` (0.75rem).
*   **Hover эффект:** При наведении карточка должна слегка менять прозрачность фона (от 0.03 до 0.06) и увеличивать Blur.
*   **Divider:** Запрещены. Используйте отступы из шкалы `spacing` (например, `10` или `12`) для визуального разделения контента внутри карточки.

### Поля ввода (Inputs)
*   Фон: `surface_container_lowest`.
*   Граница: 1px `outline_variant` (20% opacity).
*   State: При фокусе граница становится `primary` с мягким внутренним свечением.

### Дополнительные компоненты: «Status Badges»
Для бизнес-центра важны статусы (Свободно/Занято). Используйте `surface_variant` для фона бейджа и маленькую светящуюся точку (dot) соответствующего семантического цвета (`error` или `primary`).

## 6. Do's and Don'ts

| Правило | Описание |
| :--- | :--- |
| **Do** | Использовать асимметрию в Bento Grid для создания динамики. |
| **Do** | Оставлять большие негативные пространства (spacing 16-24) между смысловыми блоками. |
| **Don't** | Использовать чисто белый цвет (#FFFFFF) для текста. Всегда используйте `on_surface` или `on_surface_variant`. |
| **Don't** | Использовать резкие, темные тени. Премиальность — это мягкость и свет. |
| **Do** | Применять `backdrop-filter: blur` для всех плавающих элементов поверх контента. |
