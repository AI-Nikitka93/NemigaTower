# Support

Use the path that best matches your question so the repository stays easy to maintain.

## Quick Self-Check

- Read the root [`README.md`](./README.md) for setup, deploy, and repository scope.
- Open the live site: <https://ai-nikitka93.github.io/NemigaTower/>
- Check existing issues before opening a new one.

## Where To Ask For What

- Bug in the site, build, or deployment flow: use the bug report form in [Issues](https://github.com/AI-Nikitka93/NemigaTower/issues/new/choose)
- Feature or packaging improvement: use the feature request form in [Issues](https://github.com/AI-Nikitka93/NemigaTower/issues/new/choose)
- Security concern: follow [`SECURITY.md`](./SECURITY.md) and do not open a public issue
- General support or portfolio context question: use the public maintainer contact method listed on the GitHub profile

## Response Expectations

- This is a public demo repository, so support is provided on a best-effort basis.
- Reproducible reports with screenshots, environment details, and clear steps are much easier to help with.
